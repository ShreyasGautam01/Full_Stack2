# Experiment 2.3: Material UI Implementation

This project explores the use of **Material UI (MUI)** components within a React environment to create a standardized, accessible user interface.

## Features
* **MUI Components:** Utilizes `Card`, `TextField`, `Button`, and `Box` for a professional "Material" look and feel.
* **Flexbox Centering:** Employs an outer `Box` container to achieve perfect viewport centering.
* **Conditional Logic:** The Submit button dynamically enables or disables based on whether the "Name" input field is empty.
* **Theme Styling:** Implements custom `sx` props for background color and card dimensions.

## Tech Stack
* **React**
* **Material UI (MUI)**
* **Vite**

## Setup
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
